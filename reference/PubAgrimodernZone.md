# Details of Approved List of Agricultural Modernization Demonstration Zone.

A data set containing Approved List of Agricultural Modernization
Demonstration Zone year by year from the public site
<https://www.most.gov.cn>, see demo page
[http://nyt.hubei.gov.cn/bmdt/yw/mtksn/202208/t20220810_4256547.shtml](http://nyt.hubei.gov.cn/bmdt/yw/mtksn/202208/t20220810_4256547.smd).
with wide data format.

## Usage

``` r
PubAgrimodernZone
```

## Format

A data frame:

- year:

  integer, the official Approved year

- index:

  integer, the ordered index of list

- name:

  character, name of the Demonstration Zone

- province:

  character, province of Demonstration Zone in reduced chinese
