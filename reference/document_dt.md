Help Document the Variables List of Data Set for Package Development
This function is deprecated after v0.1.0, please use
[`help.document()`](https://huhuaping.github.io/techme/reference/help.document.md)
instead.

## Usage

``` r
document_dt(df)
```

## Arguments

- df:

  a data frame

## Examples

``` r
data(ProvinceCity)
document_dt(ProvinceCity)
#> #'   \item{index}{  }
#> #'   \item{province}{  }
#> #'   \item{city}{  }
#> #'   \item{id}{  }
#> #'   \item{province_clean}{  }
#> #'   \item{city_clean}{  }
```
