# Details of Approved List of Spot Agri-products Fresh keeping test County

A data set containing Approved List of Spot Agri-products Fresh keeping
test County year by year from the "moa-freshkeep-county" project which
public at <https://www.moa.gov.cn>, with wide data format.

## Usage

``` r
PubFreshKeepCounty
```

## Format

A data frame:

- index:

  integer, the ordered index of list

- year:

  integer, the official approved year

- name:

  character, name of the approved county

- province:

  character, province of approved county in reduced chinese
