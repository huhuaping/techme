# Details of Officer Livestock Breeding List from MOA (Xmj)

A data set containing detail information of Officer Livestock Breeding
List from the public site MOA (Xmj) <http://www.moa.gov.cn>, with wide
data format.

## Usage

``` r
PubBreedingXmj
```

## Format

A data frame:

- year:

  integer, year marked in the file name

- index:

  character, ordered index number

- province:

  character, province of origin institution

- type:

  character, livestock type

- name_origin:

  name of the origin institution

- name_change:

  name of the changed institution

- mark:

  character, marks to distinguish doc cat, such as 'cancel', 'change',
  ect. al
