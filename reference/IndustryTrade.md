# Data Set of Hi-tech Industry Trade

A data set containing Hi-tech Industry Trade from part IV of the
Yearbook of Tech, with long data format.

## Usage

``` r
IndustryTrade
```

## Format

A data frame:

- province:

  province, including nation

- year:

  year, from 2016 to 2018, and no report after 2018

- chn_block4:

  vars name, in chinese

- value:

  value of variables

- units:

  units of variables

- variables:

  vars name, by coding
