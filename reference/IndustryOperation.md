# Data Set of Hi-tech Industry Operation

A data set containing Hi-tech Industry Operation from part IV of the
Yearbook of Tech, with long data format.

## Usage

``` r
IndustryOperation
```

## Format

A data frame:

- province:

  province, including nation

- year:

  year, from 2016 to report year, and no report in year 2017

- chn_block4:

  vars name, in chinese

- value:

  value of variables

- units:

  units of variables

- variables:

  vars name, by coding
