# Data Set of Hi-tech Industry RD

A data set containing Hi-tech Industry RD from part IV of the Yearbook
of Tech, with long data format.

## Usage

``` r
IndustryRD
```

## Format

A data frame:

- province:

  province, including nation

- year:

  year, from 2016 to report year

- chn_block4:

  vars name, in chinese

- value:

  value of variables

- units:

  units of variables

- variables:

  vars name, by coding
