# Data Set of Agricultural Plastic

A data set containing Agricultural Plastic statistics from the Yearbook
of Rural, with long data format.

## Usage

``` r
AgriPlastic
```

## Format

A data frame:

- province:

  province, including nation

- year:

  year, begin with year 2010

- chn_block4:

  vars name, in chinese

- value:

  value of variables

- units:

  units of variables

- variables:

  vars name, by coding
