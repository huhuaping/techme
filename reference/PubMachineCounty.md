# Details of Approved List of Agricultural Machinery Demonstration County

A data set containing Approved List of Agricultural Machinery
Demonstration County year by year from the public site
<https://www.moa.gov.cn>, with wide data format.

## Usage

``` r
PubMachineCounty
```

## Format

A data frame:

- year:

  integer, the year of official document published

- batch:

  character, the official approved batch record in chinese

- id:

  integer, the category ID, while \<665 means "crops", 666 means
  "livestock", 777 means "facility planting"

- index:

  number, the ordered index number

- province:

  character, province names in reduced chinese

- county:

  character, county names in reduced chinese

## Source

<https://www.huhuaping.com/>
