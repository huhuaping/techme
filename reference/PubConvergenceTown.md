# Details of Approved List of Strong Agri-industry Towns

A data set containing Approved List of Strong Agri-industry Towns year
by year from the "moa-industry-convergence" project which public at
<https://www.moa.gov.cn>, with wide data format.

## Usage

``` r
PubConvergenceTown
```

## Format

A data frame:

- year:

  integer, the official approved year

- index:

  integer, the ordered index of list

- province:

  character, province of Strong Agri-industry Towns in reduced chinese

- town:

  character, name of the Strong Agri-industry Towns

- n:

  integer, numbers of towns within the province
