# Details of Approved List of National Agricultural Demonstration Park

A data set containing Approved List of National Agricultural
Demonstration Park year by year from the "moa-industry-convergence"
project which public at <https://www.moa.gov.cn>, with wide data format.

## Usage

``` r
PubConvergencePark
```

## Format

A data frame:

- index:

  integer, the ordered index of list

- year:

  integer, the official approved year

- name:

  character, name of the agri park

- province:

  character, province of agri park in reduced chinese
