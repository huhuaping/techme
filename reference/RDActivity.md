# Data Set of RD Activity

A data set containing RD Activity statistics from the Yearbook of Tech,
with long data format.

## Usage

``` r
RDActivity
```

## Format

A data frame:

- province:

  province, including nation

- year:

  year, begin with year 2010

- chn_block4:

  vars name, in chinese

- value:

  value of variables

- units:

  units of variables

- variables:

  vars name, by coding
