# Details of Approved List of National Genetic Resource Base

A data set containing Approved List of National Genetic Resource Base
year by year from the "moa-genetic-resource" project which public at
<https://www.moa.gov.cn>, with wide data format.

## Usage

``` r
PubGeneticResource
```

## Format

A data frame:

- year:

  integer, the official approved year

- batch:

  character, the batch NO.

- type:

  character, type of the base

- index:

  integer, the ordered index of list

- name:

  character, name of the base

- institution:

  character, name of the on duty institution

- province:

  character, province of approved county in reduced chinese
