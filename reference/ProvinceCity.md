# Data Set of Province and City of China

A data set containing Province and City of china from the github
repository
'wecatch/china_regions'<https://github.com/wecatch/china_regions>, with
wide data format.

## Usage

``` r
ProvinceCity
```

## Format

A data frame:

- index:

  ordered index

- province:

  name of province

- city:

  city belongs of the province

- id:

  id of city with digits 12

- province_clean:

  reduced province name in chinese

- city_clean:

  reduced city name in chinese
