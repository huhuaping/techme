# Details of Approved List of Agricultural and Rural informational Demonstration Base

A data set containing Approved List of Agricultural and Rural
informational Demonstration Base year by year from the
"moa-ruralinfo-base" project which public at <https://www.moa.gov.cn>,
with wide data format.

## Usage

``` r
PubRuralInfoBase
```

## Format

A data frame:

- year:

  integer, the official approved year

- index:

  integer, the ordered index of list

- type:

  character, type of the approved base

- institution:

  character, name of the approved base

- province:

  character, province of approved base in reduced chinese
