# Data Set of District information of China

A data set containing district information of china from the github
repository 'wecatch/china_regions'<https://github.com/eduosi/district>,
with wide data format.

## Usage

``` r
BasicDistrict
```

## Format

A data frame:

- id:

  id

- name:

  name in chn

- parent_id:

  parent id

- initial:

  first string in pinyin

- initials:

  short string in pinyin

- pinyin:

  full string in pinyin

- extra:

  extra detail

- suffix:

  suffix of district

- code:

  code of district

- area_code:

  area code of district

- order:

  order index
