# Agricultural Mechanization Services by Region

This dataset contains comprehensive statistics on agricultural
mechanization services by region, including service organizations,
personnel, investment, and output. The data is extracted from the China
Agricultural Machinery Industry Yearbook, covering all provinces and
regions in China.

## Usage

``` r
MachineService
```

## Format

A data frame:

- province:

  character. Province name, including national total.

- year:

  character. Year of the statistics.

- chn_block4:

  character. Service indicator in Chinese (e.g., "Year-end
  Organizations", "Year-end Personnel", "Service Income").

- value:

  numeric. The statistical value.

- units:

  character. Units of measurement (e.g., "organizations", "persons",
  "10,000 yuan").

- variables:

  character. Variable name in coded format.

## Source

China Agricultural Machinery Industry Yearbook, China Agricultural
Machinery Industry Association

## Details

- The dataset covers agricultural mechanization services from 2010 to
  the latest available year.

- Service indicators include organization counts, personnel numbers,
  investment amounts, and service income.

- Data structure evolved over time: separate tables from 2010-2013,
  merged tables from 2014-2017, and fully integrated tables from
  2018-2023.

- Data is in long format for easy analysis and visualization.
