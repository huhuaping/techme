# Details of Officer Livestock Standard List from MOA (Xmj)

A data set containing detail information of Officer Livestock Standard
List from the public site MOA (Xmj) <http://www.moa.gov.cn>, with wide
data format.

## Usage

``` r
PubStandardXmj
```

## Format

A data frame:

- year:

  integer, year marked in the file name

- index:

  character, ordered index number

- province:

  character, province of origin institution

- prod_name:

  character, livestock type

- com_name:

  name of the company
