# Data Set of Tech Market Trade (Push Effect)

A data set containing Tech Market Trade from the Yearbook of Tech, with
long data format.

## Usage

``` r
MarketPush
```

## Format

A data frame:

- province:

  province, including nation

- year:

  year, begin with year 2010

- chn_block4:

  vars name, in chinese

- value:

  value of variables

- units:

  units of variables

- variables:

  vars name, by coding
