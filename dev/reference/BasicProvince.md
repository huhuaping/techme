# Data Set of Basic Province information

A data set containing basic information of province and its region, with
wide data format.

## Usage

``` r
BasicProvince
```

## Format

A data frame:

- id:

  identifier with digits 1

- province:

  names of province in chinese

- region_pro:

  region of pvovince in chinese
