# Details of Approved List of Special and Advantage Industry Cluster

A data set containing Approved List of Special and Advantage Industry
Cluster year by year from the "moa-industry-convergence" project which
public at <https://www.moa.gov.cn>, with wide data format.

## Usage

``` r
PubConvergenceCluster
```

## Format

A data frame:

- index:

  integer, the ordered index of list

- year:

  integer, the official approved year

- name:

  character, name of the Industry Cluster

- province:

  character, province of Industry Cluster in reduced chinese
