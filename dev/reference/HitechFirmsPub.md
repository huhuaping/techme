# Data Set of Hi-tech Firms Numbers on Public Site

A data set Hi-tech Firms Numbers scraped from Public Site
<http://www.innocom.gov.cn/>, with wide data format. The origin scraping
script file in
"tech-report/data-raw/public-site/torch-innocom/scrap-innocom.Rmd".

## Usage

``` r
HitechFirmsPub
```

## Format

A data frame:

- index:

  index, sorting ID

- year:

  year, begin with year 2018

- province:

  provinces, in chinese

- num:

  numbers of firms
