# Data Set of Public Budget

A data set containing Public Budget statistics from the Yearbook of
Nation, with long data format.

## Usage

``` r
PublicBudget
```

## Format

A data frame:

- province:

  province, including nation

- year:

  year, begin with year 2010

- chn_block4:

  vars name, in chinese

- value:

  value of variables

- units:

  units of variables

- variables:

  vars name, by coding
