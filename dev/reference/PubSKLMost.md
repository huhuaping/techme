# Details of Stated Key Lab (SKL)

A data set containing Approved List of Stated Key Lab (SKL) year by year
from the public site <https://www.most.gov.cn>, with wide data format.

## Usage

``` r
PubSKLMost
```

## Format

A data frame:

- year:

  year of data source

- type:

  type of SKL, c('SKL','PSKL', 'FSKL')

- index:

  index

- name:

  name of SKL

- area:

  research areas of SKL

- institution:

  institution of SKL

- administrator:

  administrator of SKL

- province:

  province of SKL

- NO:

  NO of SKL

- start_year:

  start year of SKL
