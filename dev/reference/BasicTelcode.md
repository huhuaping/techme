# Data Set of Telephone area code information of China

A data set containing Telephone area code information of china from the
github repository
'wecatch/china_regions'<https://github.com/eduosi/district>, with wide
data format.

## Usage

``` r
BasicTelcode
```

## Format

A data frame:

- name:

  name of city and county

- code:

  code of city and county

- area_code:

  code of telephone

- code_province:

  code of province

- district:

  name of province
