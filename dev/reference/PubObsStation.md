# Details list of Observe Stations of MOA and MOE

A data set containing list of Observe Stations of Ministry of
Agriculture(MOA) and Ministry of Education (MOE) year by year from the
public MOA site <http://www.moa.gov.cn/> and MOE site
<http://www.moe.gov.cn/> , with wide data format.

## Usage

``` r
PubObsStation
```

## Format

A data frame:

- officer:

  name of Ministry, MOA or MOE

- year:

  integer, year of the document from Ministry site

- index:

  integer, the ordered index of list

- name:

  character, name of the agri park

- institution:

  character, name of institution hosted the stations

- administrator:

  character, name of the administrator for the institution

- province:

  character, province of agri park in reduced chinese
