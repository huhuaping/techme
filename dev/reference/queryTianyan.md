# Data Set of Institution Query from Tianyan Site

A data set containing basic info of institution including address,
telephone, url, repository
'wecatch/china_regions'<https://github.com/wecatch/china_regions>, with
wide data format.

## Usage

``` r
queryTianyan
```

## Format

A data frame:

- index:

  ordered index

- name_origin:

  origin name of institution

- name_search:

  searched name of institution

- address:

  searched address of institution

- tel:

  searched telephone of institution

- url:

  searched website url of institution

- province:

  final province of institution

- city:

  matched city of institution

- province_raw:

  matched province of institution
